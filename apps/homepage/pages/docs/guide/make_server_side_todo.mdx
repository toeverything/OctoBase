# Make a server-side Todo MVC with OctoBase

This is a tutorial on how to make a server-side Todo MVC apis with OctoBase.

Server-side Todo MVC provides the apis of adding, deleting and searching for todo data, allowing you to write a webhook to automatically add todo and synchronize it to the client.

We will be using the OctoBase Rust SDK to make the app.
